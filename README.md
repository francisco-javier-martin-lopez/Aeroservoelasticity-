# Aeroservoelasticity-

Workshops made for the course Aeroservoelasticity in Politecnico Di Milano. 

Worked on MATLAB.
In the fisrt workshop, programming a code to solve flutter problem in the Goland Wing model, with standart k & p-k methods. Furthermore, studying the influence of the addition of a rigid control surface in the flutter velocity.
For the second workshop, study of a Puma helicopter blade pitch-flap coupling. Stability map obtention and analysis of effects of pitch
link stiffness and CG of the blade.
For the last workshop, a group project featuring Alberto Rivero García and Alejandro Rivera Míguez, it was studied the Limit Cycle Oscillation apperaing in the Goland Wing Model with a free play in the control surface.
